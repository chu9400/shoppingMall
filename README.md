## 문제 해결
### - 프로퍼티 보안 
    - 개인정보 보호를 위한 프로퍼티 보안관리

    중요한 정보를 외부 파일로 분리하여 코드와 분리함으로써 보안성을 향상시킵니다.
    @PropertySource 어노테이션을 활용하여 외부 프로퍼티 파일을 로드합니다.
    중요한 정보를 변수화하여 코드에 직접 노출되지 않도록 합니다.

    예시 코드 : @PropertySource("classpath:파일경로")
